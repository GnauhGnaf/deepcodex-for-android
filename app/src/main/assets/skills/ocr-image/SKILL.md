# OCR Image Text Extraction

Run directly — no need to read this file first.

```bash
python3 /skills/ocr-image/scripts/ocr.py <file> -m grounding
```

Modes: `grounding` (tables/forms, default), `free_ocr` (simple text), `ocr_image` (word-level detail).
